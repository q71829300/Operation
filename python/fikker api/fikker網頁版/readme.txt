網頁式的fikker新增
使用統一規範:
1.將所有的證書(crt)、key都放在fikker目錄下
2.確定證書的檔名，只能域名+crt，例如:604444.com.crt，因為域名是自動取得.crt前的名稱

------------------------------------------------------------------------------------------

使用前請調整所有py腳本:

此為點開新增源站的位子，可先測試腳本的位置，如果不對可在調整
pyautogui.click(572,932)   

此為新增源站的鈕，，可先測試腳本的位置，如果不對可在調整
pyautogui.click(733,933)

此預設開啟的fikker往頁為測試站的網頁，使用時請調整至正式站
driver.get("http://fikker_ip/fikker/")

------------------------------------------------------------------------------------------
檔案說明:

/domain/domain.txt - 用於多個域名對上同一個證書及key

/source/source.txt - 用於新增域名時要加的源站IP


腳本用途說明:
------------------------------------------------------------------------------------------

/py/fikker.py - 用於單一個域名對應一個證書使用

使用方式:

1.在source.txt新增源站

------------------------------------------------------------------------------------------

/py/fikker_m_d.py - 用於多個域名對應一個證書使用

使用方式:

1.在source.txt新增源站
2.在domain.txt新增域名，請新增時使用換行的方式
範例:
www.com.tw
ccc.com.tw
ttt.com.tw
ppp.com.tw

------------------------------------------------------------------------------------------

/py/fikker_sr.py - 用於新增新CDN IP使用

使用方式:

1.在下指令時，請帶 -i 參數在加上IP，如果想提示請帶 -h 參數
範例:
python -u "c:\Users\user\Desktop\fikker_sr.py" -i 192.168.0.1
